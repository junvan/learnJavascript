* JavaScript 的类型有数字、字符串、布尔值、函数和对象。还有 undefined 和 null ，以及数组、日期和正则表达式。

##### 操作符
|算数操作符|描述|
|:--:|:--:|
|+|加法|
|-|减法|
|*|乘法|
|/|除法|
|%|取余|
|++|递增|
|--|递减|

|赋值操作符 | 描述|
|:--:|:--:|
|=|赋值|
|+=|加/赋值  (x += y) == (x = x + y)
|-=|减/赋值  (x -= y) == (x = x - y)
|*=|乘/赋值  (x *= y) == (x = x * y)
|/=|除/赋值  (x /= y) == (x = x / y)
|%=|取余/赋值  (x %= y) == (x = x % y)
